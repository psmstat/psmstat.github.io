Statistical image comparison and monitoring

  • Formulating statistical methodologies for image comparison and monitoring in real-world applications across diverse scientific disciplines.
  • Intending to address fundamental challenges in image monitoring by unifying state-of-the-art techniques from both domains: image processing and statistical process control.
  • Aiming to lay the groundwork for more robust, scalable, and interpretable image-based monitoring systems and advancing the field towards broader practical impact.
  • Selected publications:

    • Roy, A.* and Mukherjee, P.S. "Image Comparison Based On Local Pixel Clustering", Technometrics, 2024, 66(4), 495--506. [Link][Computer Codes]
    * doctoral student

Image registration using jump regression analysis

  • Aiming to make the first stride towards widening the scope of rigid-body image registration to include zooming or scaling under the framework of jump regression analysis.
  • The goal is to develop image registration methods that can perform well on a wide range of situations so that they can be useful in various downstream applications, such as image comparisons, image monitoring, and so forth.
  • Intending to develop ideas which are adaptable and may serve as a foundation for developing a broader class of techniques in data science, including methods for nonlinear image registration.
  • Selected publications:
    • Das, S.* and Mukherjee, P.S. "Image Registration for Zooming Using Similarity Matching", Journal of Data Science, 2024, 22(4), 558--574. [Link] [Computer Codes]
    • Das, S.*, Roy, A.* and Mukherjee, P.S. "Image registration for zooming: A statistically consistent local feature mapping approach", Stat, 2024, 13(1), e664. [Link] [Computer Codes]
    * doctoral student

Jump regression analysis using regression trees

  • Working on a regression treeâ€“based framework based on Oblique-axis Regression Trees (ORT) for estimating discontinuous regression functions in finite-dimensional spaces, and demonstrates its potential through effective image denoising.
  • Supervising a doctoral student who is rethinking the assumptions imposed on the underlying regression function, so that the resulting theoretical analysis attains a level of structural simplicity that departs from much of the existing regression tree literature.
  • Extending the ORT framework beyond static estimation, and intending to develop a method for monitoring drift patterns in image sequences. Although gradual temporal variations, commonly referred to as drifts, are ubiquitous in sequential imaging data, principled statistical approaches for drift monitoring remain largely unexplored.
  • Selected publications:
